avrdude toolchain setup (Windows 10):
	1) extract avrdude-win-64bit.zip to C:\Program Files.
	2) add C:\Program Files\avrdude to PATH
		- open Control Panel
		- go to System and Security 
		- choose System
		- on the left, choose Advanced system settings
		- select Environment Variables
		- on the top panel, select Path on the left and hit Edit...
		- choose Browse on the right and select the path, C:\Program Files\avrdude
		- hit ok
	
	3) install drivers for Pocket AVR Programmer: 
		- follow this tutorial: https://learn.sparkfun.com/tutorials/pocket-avr-programmer-hookup-guide/installing-drivers
